About

We’re a fictitious music streaming service with a real website and API. Built for educational purposes, you can use us to learn how to extract web data through web scraping and APIs – without the hassle of spitting through restrictive terms of use or being blocked.

Our service is inspired by Zyte’s Books to Scrape, which you can also use to learn web scraping. Yet, we also provide an API – plus, our entire code base is open, so you extend the site and API or even change the simulated data.

Our service is currently used to teach web scraping to students at Tilburg University. We’ve been developed by Tilburg Science Hub, Tilburg University’s platform for teaching and applying open science.

How our service works

Our service consists of three components: the front-end website written in Flask, an API backend written in FastAPI, and a data simulation (remember: all data, except artist and song names, is entirely made up!) written in R.

The only real part in our data is the artist and song names, for which we would like to credit Mahkaila, a GitHub user. Oh, and there’s the photo of the trumpet player on our landing page – he’s a real musician from Caracas, Venezuela – thanks for sharing your picture!

We’re currently deployed on Azure. Our code base is available at GitHub. Head over to our repository to find out how you can contribute to it.
